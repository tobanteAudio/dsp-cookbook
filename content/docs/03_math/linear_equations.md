# Linear Equations

## Example A

{{< katex display >}}
x + 2y = 5
{{< /katex >}}

{{< katex display >}}
3x + 4y = 6
{{< /katex >}}

{{< katex display >}}
A =
\begin{pmatrix}
1 & 2 \\
4 & 3
\end{pmatrix}

\quad

b =
\begin{pmatrix}
5 \\
6
\end{pmatrix}
{{< /katex >}}

{{< katex display >}}
x = A^{-1} \cdot b
{{< /katex >}}
